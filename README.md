# ZPAD_LABS_2
Лабораторні роботи з навчальної дисципліни «ЗПАД». Виконав студент групи ФБ-46 Ільченко Владислав.

Мета роботи:
Набуття навичок роботи з бібліотекою Pandas, обробка великих масивів даних (CSV/TXT) та реалізація алгоритмів аналізу екологічних індексів (VHI).

Опис виконання:
1. Частина 1 (Аналіз VHI):
   - Реалізовано механізм завантаження та об'єднання 27 CSV-файлів з індексами VHI для кожної області України.
   - Створено словник для заміни ідентифікаторів областей на їхні назви за українським алфавітом.
   - Написано функції для пошуку екстремальних (VHI < 15) та помірних (15 <= VHI <= 35) посух за вибраний рік.
   - Передбачено генерацію локальних даних у разі недоступності сервера NOAA (помилка 404).

2. Частина 2 (Аналіз енергоспоживання):
   - Проведено очищення та фільтрацію датасету споживання електроенергії (близько 2 млн записів).
   - Виконано вибірку за критеріями: активна потужність > 5 кВт, напруга > 235 В, сила струму 19-20 А.
   - Проведено випадкову вибірку 15 000 записів та порівняння рівнів споживання (`Sub_metering`).

Висновок:
У ході роботи було опановано методи фільтрації, очищення даних та роботу з часовими рядами у Pandas. Програма працює коректно та виводить структурований звіт у форматі Jupyter Notebook.
